# 用户故事地图 (User Story Map)

## 1. 概述
本用户故事地图以“业务人员生成数据看板”为核心叙事线索，将用户活动分解为具体任务和故事，并映射到开发版本中。

## 2. 结构视图

### Backbone (用户活动流)
**1. 接入数据** -> **2. 定义指标** -> **3. 探索与配置** -> **4. 组装看板** -> **5. 输出交付**

---

### 3. 详细拆解

#### Activity 1: 接入数据 (Data Ingestion)
| 任务 (Tasks) | Story (用户故事) | 优先级 | 版本 |
| :--- | :--- | :--- | :--- |
| **上传文件** | 作为用户，我想要上传 CSV/Excel 文件，以便导入我的业务数据。 | P0 | MVP |
| **检查数据** | 作为用户，我想要预览上传数据的前几行，以确保解析正确。 | P0 | MVP |
| **处理缺失** | 作为用户，我希望系统自动填充缺失值（0或Unknown），以免报错。 | P1 | V1.0 |
| **类型修正** | 作为用户，我想要手动修正列的数据类型（如将ID转为文本），以防识别错误。 | P2 | V2.0 |

#### Activity 2: 定义指标 (Metric Definition)
| 任务 (Tasks) | Story (用户故事) | 优先级 | 版本 |
| :--- | :--- | :--- | :--- |
| **选择指标** | 作为用户，我想要从下拉框选择核心指标列，以便进行分析。 | P0 | MVP |
| **设定时间** | 作为用户，我想要指定时间列，以便生成趋势图。 | P0 | MVP |
| **自定义公式** | 作为用户，我想要通过公式（如 A/B）创建新指标，以便分析计算字段。 | P1 | V1.0 |
| **处理特殊名** | 作为用户，我希望公式支持包含空格的列名，因为我的表头通常有空格。 | P1 | V1.0 |

#### Activity 3: 探索与配置 (Explore & Configure)
| 任务 (Tasks) | Story (用户故事) | 优先级 | 版本 |
| :--- | :--- | :--- | :--- |
| **查看趋势** | 作为用户，我想要看到核心指标随时间的趋势图，以了解业务走势。 | P0 | MVP |
| **维度筛选** | 作为用户，我想要筛选特定的维度（如渠道、地区），以进行细分分析。 | P1 | V1.0 |
| **查看分布** | 作为用户，我想要看各维度的占比（饼图/玫瑰图），以了解结构。 | P0 | MVP |
| **查看排名** | 作为用户，我想要看 Top N 的排名，以找出头部贡献者。 | P1 | V1.0 |
| **转化分析** | 作为用户，我想要生成漏斗图，以分析各环节转化率。 | P1 | V1.0 |
| **异常检测** | 作为用户，我希望图表自动标记异常点，以便我快速发现问题。 | P1 | V1.0 |

#### Activity 4: 组装看板 (Assemble Dashboard)
| 任务 (Tasks) | Story (用户故事) | 优先级 | 版本 |
| :--- | :--- | :--- | :--- |
| **添加图表** | 作为用户，我想要将当前配置好的图表保存到看板区，以免丢失配置。 | P0 | MVP |
| **删除图表** | 作为用户，我想要移除不满意的图表，以保持看板整洁。 | P0 | MVP |
| **调整顺序** | 作为用户，我想要拖拽调整图表顺序，以梳理叙事逻辑。 | P2 | V2.0 |
| **自动描述** | 作为用户，我希望系统自动生成图表的文字解读，以节省我写文案的时间。 | P2 | V2.0 |

#### Activity 5: 输出交付 (Export & Deliver)
| 任务 (Tasks) | Story (用户故事) | 优先级 | 版本 |
| :--- | :--- | :--- | :--- |
| **预览全貌** | 作为用户，我想要预览最终生成的单页效果，以进行最后检查。 | P0 | MVP |
| **导出HTML** | 作为用户，我想要下载一个独立的 HTML 文件，以便离线展示。 | P0 | MVP |
| **导出CSV** | 作为用户，我想要导出处理后的汇总数据，以便做二次加工。 | P1 | V1.0 |
| **导出图片** | 作为用户，我想要导出高清 PNG 图片，以便插入 PPT。 | P2 | V2.0 |

## 4. 版本发布规划
- **Release 1 (MVP)**: 聚焦于 "Activity 1, 2, 3(基础), 4(基础), 5(HTML)" 的 P0 故事。
- **Release 2 (V1.0)**: 增加 "Activity 2(公式), 3(高级图表/筛选), 5(CSV)" 的 P1 故事。
- **Release 3 (V2.0)**: 增加 "Activity 4(智能描述/排序), 5(图片导出)" 的 P2 故事。
